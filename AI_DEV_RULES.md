# AI Development Rules for Chase AI Habitat

These rules apply to any AI assisting with this project.

1. Always provide FULL FILE REPLACEMENTS when suggesting code changes.
2. Do NOT provide partial snippets unless explicitly requested.
3. Assume the user will copy/paste the entire file.
4. Maintain compatibility with the current architecture.
5. Avoid introducing breaking changes without explanation.

Reason:
The project is frequently updated manually and partial patches cause merge errors.
